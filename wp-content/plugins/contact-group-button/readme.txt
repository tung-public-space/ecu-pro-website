== Description ==

Contact Group Button is a plugin that allows you to enter your contact information such as: Phone number, facebook messages link, zalo number, sms number.

Display the website with the first button, then when the customer clicks on this button will display the rest of your information allowing customers to contact you immediately.

The plugin allows editing position display, contact information, color buttons, color effects.

Plugin with powerful spreading effect to create attention to customers and create sympathy with them.

**Features**

Customize display on-screen destop or mobile and table or both.
Customize the location of the contact group button anywhere you want.
Customize the background color button.
Customize button effect color.
Customize your phone number.
Customize your zalo number.
Customize your link facebook messages.

== Installation ==

1. Upload the entire 'contact-group-button' folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the Settings->Contact Group Button screen to configure the plugin.
